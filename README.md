** What is Fortune Jobs AI?**

Fortune Jobs AI is an AI-powered job matching platform built from scratch with zero coding experience.

You type a job title → AI scrapes real jobs from Indeed → Claude AI picks the best match → Results appear with salary, location and Apply Now link.


**How It Works**

1. User types a job title on the website
2. Make.com webhook receives the search
3. Claude AI extracts the clean job title
4. Apify scrapes real live jobs from Indeed
5. Claude AI picks the single best matching job
6. Result is saved to Supabase database
7. Website displays the job card automatically


**Tech Stack**

| Tool | Purpose |
|------|---------|
| Claude AI (Anthropic) | AI job matching |
| Apify | Real-time Indeed scraping |
| Make.com | Automation pipeline |
